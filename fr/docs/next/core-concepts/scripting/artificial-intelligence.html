Intelligence artificielle

Comment l'IA fonctionne dans nanos world

caution

🚧 Cette page est en cours de rédaction.

Every Character spawned without a Player possessing it will automatically be possessed by an AI Controller. Cela signifie que vous pourrez utiliser les méthodes de Personnage telles que :MoveTo() et :LookAt().

In nanos world, AI as well as Physics are things distributed through Clients to be calculated and shared with the other Players (please refer to Network Authority), which means the AI will only work if there is a Player connected to the server.

info

Si vous utilisez la fonction :MoveTo() sur un PNJ et qu'aucun joueur n'est connecté, le PNJ commencera à marcher quand un joueur rejoindra le serveur.

Mesh de navigation​

Afin que les PNJs puissent trouver le chemin à parcourir, la map doit avoir un NavMesh configuré.
